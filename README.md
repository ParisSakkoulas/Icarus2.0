# UniGrade-Manager
The focus of this project was the design and implementation of a web application that enables the management of courses and their respective grades.It is designed to be utilized by both university professors and students, allowing them to manage courses and grades efficiently. Moreover, the application is accessible to anonymous visitors, enhancing its inclusivity and user reach.This project, named 'UniGrade Manager,' represents a significant advancement over the existing Icarus Project. It offers enhanced features and functionalities, providing a more comprehensive and user-friendly platform for managing courses and grades. With its intuitive interface and advanced capabilities.
